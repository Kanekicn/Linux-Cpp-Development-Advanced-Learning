## 12.1 操作符重载

- 关键词 `operator` 

```C++
Type operator Sign(const Type p1, const Type p2){
  	Type ret;
  
  	return ret;
}
```

解法2:利用操作符重载

- 操作符左右的两个参数
- 将该函数声明为类的友元函数

```C++
Complex operator + (const Complex& p1, const Complex& p2)
{
    Complex ret;

    ret.a = p1.a + p2.a;
    ret.b = p1.b + p2.b;

    return ret;
}
```

## 12.2 类中操作符重载

- 编译器优先在成员函数中寻找操作符重载函数

```C++
class Type{
  public:
  		Type operator Sign(const Type& p){
        	Type ret;
        
        	return ret;
      }
}
```
